Project Overview

This project explores the Free Application for Federal Student Aid (FAFSA) application trends across 5 quarters during the 2023–2024 academic year. Our goal is to uncover institutional and regional patterns that can inform education policy and outreach strategies. Tasks involved data cleaning, exploratory data analysis, visualization (with both D3.js and Altair), and web-based storytelling.

Data Description

The dataset (cleaned.csv) contains 6108 U.S. institutions, collected from the Federal Student Aid Data Center. Key columns include:

  • School Information: Name, State, Zip Code, Type
  • FAFSA Applications: Dependent and Independent counts for Q1–Q5
  • Quarterly Totals per school and state
